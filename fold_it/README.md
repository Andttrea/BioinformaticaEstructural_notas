# Reporte de la Actividad 

En esta actividad se exploró el uso de **Foldit**, un videojuego diseñado para resolver problemas de plegamiento de proteínas mediante la manipulación interactiva de estructuras 3D.

**Foldit** permite a los jugadores ajustar manualmente la conformación de proteínas, modificando elementos como hélices alfa, láminas beta y cadenas laterales. El objetivo es encontrar la estructura nativa más estable, minimizando la energía del sistema basándose en principios bioquímicos reales.

Puede descargar el juego en el siguiente enlace: [Foldit](https://fold.it/play)
